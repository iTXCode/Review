### 红黑树

- [ ] cur,parent,uncle

#### uncle存在且为红

- parent,uncle变黑
- gparent:红
- cur=gparent:继续向上调整
#### uncle存在且为黑/不存在

- 需要旋转
- cur,parent在同一边的时候需要进行单旋
- cur,parent不在同一边的时候进行双旋:通过第一次旋转转换为第一种情况 -->单旋
- parent:黑
- gparent:红
- cur=parent:继续进行调整
